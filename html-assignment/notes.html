Important HTML Tags

The "p" Tag
The "p" tag is used to create a paragraph.
The "strong" Tag
The "strong" Tag is used to make text bold.
The "ul" Tag
The "ul" tag allows for the creation of an unordered list.
The "article" Tag
The "article" tag represents an independent area for composition.
The "nav" Tag
The "nav" tag creates a space to hold navigational links for the website.
